# MAT File Analyzer

פרויקט לניתוח קבצי .mat

## התקנה

1. צור סביבה וירטואלית:
```bash
python -m venv venv
```

2. הפעל את הסביבה הוירטואלית:
- Windows:
```bash
.\venv\Scripts\activate
```
- Linux/Mac:
```bash
source venv/bin/activate
```

3. התקן את הדרישות:
```bash
pip install -r requirements.txt
```
